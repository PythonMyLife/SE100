# 学积分计算：一个简单的前端应用

## 设计目的：计算学积分

## 技术栈：

html+Vue.js + node.js + npm + mocha + chai

## 应用功能

 1. 增加和删除课程
 2. 随着增加和删除课程计算对应的学积分
 3. 删除时弹出询问框
